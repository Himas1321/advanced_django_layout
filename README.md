# Player

Пример подключения плеера

## Как установить

1. Рекомендуется использовать [virtualenv/venv](https://docs.python.org/3/library/venv.html) для изоляции проекта.
2. Для запуска необходима версия [Python 3.12.*](https://www.python.org/downloads/).
3. Скачайте код.
4. Затем используйте `pip` (или `pip3`, есть конфликт с Python2) для установки зависимостей:
```sh
pip install -r requirements.txt
```

## Запуск

Для запуска сайта локально у вас уже должен быть установлен Python.
В терминале пропишите
```sh
livereload
```

После этого переходите по ссылке [http://127.0.0.1:35729/](http://127.0.0.1:35729/), вы увидите плеер.
